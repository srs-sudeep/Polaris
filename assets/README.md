# Assets Folder

Place your application icons here:

- `icon.png` - Application icon (PNG format, 256x256 or larger)
- `icon.ico` - Windows icon (for Windows builds)
- `icon.icns` - macOS icon (for macOS builds)

## Creating Icons

You can create icons using online tools or image editing software:

1. Create a 256x256 (or larger) PNG icon
2. For Windows: Convert to .ico using online converters or ImageMagick
3. For macOS: Convert to .icns using `iconutil` or online converters

If icons are not provided, the application will use Electron's default icon.
